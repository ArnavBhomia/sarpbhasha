* **Clipboard:** `pyperclip`
* **Packaging:** `PyInstaller`

## How to Run

1.  Download the standalone .exe for Windows from the [Releases page](https://github.com/ArnavBhomia/sarpbhasha/releases).
2.  OR clone the repository and run the script locally:

```bash
# Clone the repo
git clone [https://github.com/ArnavBhomia/SarpBhasha.git](https://github.com/ArnavBhomia/SarpBhasha.git)
cd SarpBhasha

# Install dependencies
pip install -r requirements.txt

# Run the app
python SarpBhasha.py
